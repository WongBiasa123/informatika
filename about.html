    Tentang Saya

    Perkenalkan nama saya Ahmad Hasby Maulana, Saya dari kelas X TKJ 1, SMKN 1 Kota Kediri. Saya tinggal di Ds Sendang Kec. Banyakan Kab. Kediri. Hobi saya adalah Ngoding, Saya berharap kedepannya saya menjadi seorang programmer.

    Tentang Website

    Website ini saya buat dengan tujuan sebagai media penyampaian hasil survei mapel Informatika yang telah saya lakukan. Saya memilih menggunakan media website karena menurut saya website bisa diakses kapanpun dan dimanapun tanpa harus download terlebih dahulu selagi ada akses internet dan alasan saya yang kedua adalah hobi saya ngoding, jadi saya ingin meperbanyak pengalaman ngoding saya.

    Survei ini saya beri judul "Kendaraan Saat Bepergian", Alasan saya memilih survei ini karena masih banyak orang yang binggung dalam memilih kendaraan yang digunakan untuk bepergian keluar kota / mudik, Oleh karena itu saya ingin memberi sedikit rekomendasi dengan melakukan survei ke beberapa orang.

    Sebelumnya saya minta maaf jika desain atau tampilan website ini kurang menarik, karena saya lebih minat ke Backend daripada Frontend jadinya tampilannya tidak terlalu menarik, oleh karena itu saya mohon saran dan kritikannya, kalian bisa langsung menghubungi saya maupun tinggalkan pesan dikolom komentar artikel ini. Website ini saya buat dengan konsep seperti artikel atau blog denagn menggunakan HTMl, CSS dan sedikit Java Script.
